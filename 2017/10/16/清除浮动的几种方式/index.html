清除浮动的几种方式

清除浮动

  • 方法一,给父级加高

    height: 100px;

  • 方法二,在浮动元素后面添加一个新的元素，添加属性clear：both

  • 方法三,给其父级添加

    overflow: hidden;

  • 方法四,给其父级添加

    overflow: auto;

  • 方法五,让其父级一起浮动，但会产生新的浮动问题 不建议使用

  • 方法六,父级div定义为

    display：table

  • 方法七,添加伪元素来清除浮动,常用这来清除浮动

    .div1::after{
       content:"";
       display: block;
       clear:both;
   }
   .div1::after{
       zoom:1;
   }
