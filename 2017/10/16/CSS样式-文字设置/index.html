CSS样式-文字设置

今天主要是对文字进行设置，下面我为大家逐一介绍一下：

*背景颜色
英语单词，16进制，hsl,rgb

  • 尺寸单位
    px,百分比 最小min-width,min-height,最大max-width,max-height,vw(浏览器的宽度的百分比) vh
  • color：文字颜色
  • font-size
    font-size:文字字号 可以设置百分比 倍数em（相对于父级） 倍数rem（相对于根元素)(一旦像素大于12时，就按照像素的大小乘以倍数，一旦小于12像素就按照12像素）
  • text-indent
    text-indent:首行缩进 (支持中文）使用em不需考虑文字的大小
  • font-weight
    font-weight:字重 500~900=bold加粗，bolder更粗（无效果，默认解析为bold）400=normal（正常状态）100~300lighter（更细）
  • font-style
    font-style:斜体设置italic（斜体),normal(正常)
  • font-family
    font-family字体设置,前面字体设置，可以设置多个字体，中间以逗号隔开，后面设置字体类型
  • line-height
    line-height 文字居中
  • text-align
    文字对齐text-align
  • letter-spacing
    letter-spacing 字符间距(字与字之间）单词不换行，中文自动换行
  • word-spacing
    word-spacing 词间距 (不支持中文）
  • 文字修饰
    文字修饰 text-decoration:overline,line-through,underline
  • vertical-align
    vertical-align 竖直方向设置（极限对其）
