jQuery获取元素以及CSS样式

获取元素

$(“”)里面给的是字符串-选择器
下面举几个小例子：

console.log($(".item"));
console.log($("div"));
console.log($("input[type=radio]"));
console.log($("div:not(.item)"));

first 是第一个P标签

console.log($("p:first"));
console.log($("p:nth-child(1)"));
$("p").css("color","red");

odd基数索引 even偶数索引

$("p:odd").css("color","red");
$("p:even").css("color","red");

eq(2)从 0 开始计数

$("p:eq(2)").css("color","red")

gt(2)从 0 开始计数 2之后的变换

$("p:gt(2)").css("color","red")

删除元素

* remove这个方法不会把匹配的元素从jQuery对象中删除，因而可以在将来再使用这些匹配的元素。但除了这个元素本身得以保留之外，其他的比如绑定的事件，附加的数据等都会被移除。
* detach这个方法不会把匹配的元素从jQuery对象中删除，因而可以在将来再使用这些匹配的元素。与remove()不同的是，所有绑定的事件、附加的数据等都会保留下来。

 $("p:nth-child(3)").css("color", "yellow").click(function(){
         console.log("123");
     }).detach().appendTo(document.body);
     $("p:nth-child(3)").css("color", "yellow").click(function(){
         console.log("123");
     }).remove().appendTo(document.body);

修改/获取CSS样式

css和style 属性都可以获取到

console.log($(".content").css("color"));
console.log($("p:eq(1)").css("color"));
console.log($("p").eq(1).css("color"));

设置样式第一种样式 

$("p").css("color","pink");

设置样式第二种样式

$("p").css({
        color:"red",
        border:"1px solid green"

    } )

设置样式第三种样式

var cArr=["yellow","red","green","purple","cyan","pink"]
$("p:last").css("color",function(index,oldvalue){
    console.log(index,oldvalue);
    return cArr[index];
})
$("p").css({
    width:function(index,oldValue){
        return 100*(index+1);
    },
    height:function(){
        return 50;
    },
    "text-align":function(){
        return "center";
    },
    "line-height":function(){
        return 50+"px";
    }
})
